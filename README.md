# TaskFlow Pro – Employee Management System

## A. Project Title & Description

**Project Title:**
TaskFlow Pro – Employee Management System

**Description:**
TaskFlow Pro is a simple web application developed as an individual assignment for the IMS566 ADVANCED WEB DESIGN DEVELOPMENT AND CONTENT MANAGEMENT. This project was developed to demonstrate a responsive web application interface using Bootstrap 5 and JavaScript. It includes a login page, dashboard, employee page, department page, and performance page. The project focuses on front-end development only, so the login system uses hardcoded credentials without a database.

---

## B. Features Included

- User Login Authentication (Hardcoded)
- Dashboard with Employee Summary Cards
- Responsive Navigation Bar
- Employee Directory Page
- Department Directory Page
- Performance Report Page
- Data Visualization using Chart.js
- Responsive Design
